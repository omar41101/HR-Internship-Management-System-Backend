# HR & Internship Management System (HRcoM)

This is the backend repo of HRcoM, the DotJcoM's web platform that centralizes the HR processes and more.

---

## Running the code

### 1. Clone the repository
  Run `npm i` to install the dependencies.

  Run `npm run seed` to insert the initial documents (user roles, departments, leave types, ....).

  Run `npm run dev` to run the backend part of the app.
